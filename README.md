# 🌾 Krushi Setu – Java OOP & GUI Project

## 📌 Overview

**Krushi Setu** is a Java-based application developed using **Object-Oriented Programming (OOP)** concepts and **Swing GUI**.
It is designed to act as an **agricultural coordination platform** where farmers, officials, and market syndicates interact to improve planning and reduce overproduction.

---

## 🎯 Problem Statement

In agriculture, lack of coordination leads to:

* Overproduction 📉 (price crash)
* Underproduction 📈 (shortage)
* Food wastage 🚫

**Krushi Setu solves this by enabling pre-harvest data collection and analysis.**

---

## 💡 Solution

The system allows:

* 👨‍🌾 Farmers to register crop details (land, crop type, quantity)
* 🏛️ Officials to monitor farmer data
* 📊 Syndicate members to analyze supply and detect overproduction

---

## 🧠 Key Features

* ✅ Role-based system (Farmer / Official / Syndicate)
* ✅ GUI interface using Java Swing
* ✅ Crop data entry and storage
* ✅ Real-time data viewing
* ✅ Supply analysis with overproduction warning ⚠
* ✅ Clean OOP implementation

---

## 🚀 How to Run

1. Save the file as:

```
KrushiSetuGUI.java
```

2. Open terminal in the folder

3. Compile:

```
javac KrushiSetuGUI.java
```

4. Run:

```
java KrushiSetuGUI
```
